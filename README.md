# 📱 iPhone Products Showcase

A responsive front-end web page that displays different iPhone models with interactive features using **HTML, CSS, and Bootstrap**.

---

## ✨ Features

- 📱 Responsive product cards using Bootstrap grid system  
- 🎨 Clean and modern UI design  
- 🔽 Interactive collapsible sections to show different phone colors  
- 🧭 Navigation between pages (Back to Home link)  
- 🖼️ Image-based product display  
- ⚡ Smooth hover effects on cards  
- 📦 Organized product layout for multiple devices  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons
